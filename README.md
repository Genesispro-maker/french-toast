# french-toast

## French-Toast is a simple Toast playground i Built in react while learning advanced react component API design

> Built From Scratch by [me](https://github.com/Genesispro-maker)


## Built With

French-Toast was built from scratch with the following technologies:

- vite
- Nodejs
- Lucide-react
- typescript

## Prerequisites

- Nodejs (v14 or higher) [install](https://nodejs.org/en/download/)
- vite [install](https://vite.dev/)
- 
## Installation
1. Clone the repo

```bash
git clone https://github.com/Genesispro-maker/french-toast.git
```

2. Install dependencies

```bash
# npm
cd french-toast && npm install

# yarn
cd french-toast && yarn
```

3. Run Files

```
npm run dev
```
